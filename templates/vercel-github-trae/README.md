# Vercel–GitHub–Trae Deployment Template

Use this template to bootstrap new projects with a proven configuration.

## Contents
- `vercel.json` – optional overrides for builds
- `.env.example` – environment variable template
- `checklist.md` – deployment checklist

## Usage
1. Create a new repository from this folder.
2. Copy project files and adapt `.env.example` to your environment.
3. Connect the repo to Vercel and configure environment variables for all environments.
4. Push to `main` to auto-deploy.
